EUC-JP Text Codec

The EUC-JP codec provides conversion to and from EUC-JP, the main legacy encoding for Unix machines in Japan.

The environment variable UNICODEMAP_JP can be used to fine-tune the JIS, Shift-JIS, and EUC-JP codecs. The ISO 2022-JP (JIS) Text Codec documentation describes how to use this variable.
